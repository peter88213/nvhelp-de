Optionen

With Extras > Ablenkungsfreies Schreiben-Plugin Einstellungen, you can open the Optionen dialog box.

Änderungen übernehmen

When switching between sections, or when leaving the distraction free writing mode, changes of the section content will be applied to the novelibre project.

If the Nach Bestätigung fragen checkbox is checked, you will be asked for confirmation.

Bemerkung

Applying changes does not save your current project on disk, but update it in memory, und set the change notification in the novelibre footer bar. To save changes while in distraction-free writing mode, press Strg-S.

Wortzählung

By default, word count is updated manually by pressing the F5 key.

If the Laufende Aktualisierung checkbox is checked, the word count can be updated “live”, i.e. just while entering text.

Bemerkung

Live updating the word count is resource intensive und may slow down the program when editing big sections. This is why it’s disabled by default.

Farbkombinationen

You can select a color set (theme) by clicking on the button below the preview.
