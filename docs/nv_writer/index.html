English

nv_writer

Benutzerhandbuch

Diese Seite gilt für die neueste Ausgabe von nv_writer. Sie können sie mit Hilfe > Ablenkungsfreies Schreiben-Plugin Online-Hilfe öffnen.

Das Plugin fügt einen Eintrag Im ablenkungsfreien Modus schreiben sowohl zum novelibre-Abschnitt-Menü und dem Abschnitts-Kontextmenü hinzu, sowie einen Eintrag Ablenkungsfreies Schreiben-Plugin Einstellungen zum Extras-Menü und einen Eintrag Ablenkungsfreies Schreiben-Plugin Online-Hilfe zum Hilfe-Menü. Die Werkzeugleiste erhält eine -Schaltfläche.

Sie können Abschnitt für Abschnitt eintippen, wie mit einer Schreibmaschine oder einer DOS-Textverarbeitung. Das geht viel schneller als mit OpenOffice/LibreOffice. Sie können Text eingeben und hervorheben, und Sie können neue Abschnitte erzeugen. Weder gibt es eine Rechtschreibprüfung, noch können Sie neu eingegebenem Text Fremdsprachen zuweisen. Formatierungen und Sprachzuweisungen, die Sie bereits mit OpenOffice/LibreOffice erzeugt haben, bleiben jedoch erhalten. Damit können Sie Ihren ersten Entwurf vollständig in novelibre herunterschreiben, um dann das Manuskript mit OpenOffice oder LibreOffice Writer zu überarbeiten. Die Wortanzahl wird angezeigt, so dass Sie nv_writer auch für einen „Wortsprint“ im NaNoWriMo-Stil zwischendurch benutzen können.

Den ablenkungsfreien Schreibmodus starten

  1. Wählen Sie den Abschnitt aus, mit dem Sie beginnen möchten (optional).

    • Wenn beim Aufrufen des Schreibmodus kein bearbeitbarer Abschnitt ausgewählt ist, wird der letzte Abschnitt geladen und die Cursorposition wiederhergestellt. Wenn der letzte Abschnitt unbekannt ist, wird der erste bearbeitbare Abschnitt geladen.

    • Wenn beim Aufrufen des Schreibmodus der zuletzt bearbeitete Abschnitt ausgewählt ist, wird die Cursorposition wiederhergestellt.

    • Wenn beim Aufrufen des Schreibmodus ein anderer bearbeitbarer Abschnitt ausgewählt ist, wird der Cursor an den Anfang gesetzt.

  2. Um in den ablenkungsfreien Schreibmodus zu gelangen, können sie entweder

    • F11 drücken, oder

    • mit der rechten Maustaste auf den ausgewählten Abschnitt klicken und Im ablenkungsfreien Modus schreiben auswählen, oder

    • Abschnitt > Im ablenkungsfreien Modus schreiben auswählen, oder

    • Auf die -Schaltfläche auf der Werkzeugleiste klicken.

Wichtig

Um ein Durcheinander zu vermeiden, wird der ablenkungsfreie Schreibmodus nicht starten, wenn entweder

  • ein Manuskript zur Bearbeitung existiert, oder

  • das Projekt gesperrt ist.

Wenn Sie also ein normales Manuskript oder ein Manuskript für eine fremde Textverarbeitung haben, sollten Sie es verwerfen, nachdem Sie Ihr novelibre-Projekt aktualisiert haben.

Den ablenkungsfreien Schreibmodus beenden

  • Drücken Sie F11 oder Alt-F4 (Windows) oder Strg-Q (Linux), um die Bearbeitungssitzung zu beenden.

Hinweis

Im Gegensatz zu Writer ersetzt der ablenkungsfreie Editor Anführungszeichen, Bindestriche usw. nicht automatisch. Typografische Anpassungen müssen daher in einem Korrekturprozess im regulären Manuskript mit Writer vorgenommen werden. Ich empfehle zu diesem Zweck eine der curly-Erweiterungen.
